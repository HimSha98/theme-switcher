# Theme Switcher

A simple React-based theme switcher that allows users to toggle between light and dark modes.

## 🚀 Tech Stack
- React
- Vite
- JavaScript
- CSS

## ✨ Features
- Light/Dark mode toggle
- Smooth UI experience
- Fast performance with Vite
